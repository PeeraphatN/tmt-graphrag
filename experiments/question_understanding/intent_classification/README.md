# Intent Classification

This area is the target home for intent-classification experiments.

Planned subareas:

- `data/`: benchmark datasets and builders
- `embedding_model_selection/`: embedding benchmark runs
- `intent_structure_fci_hic/`: FCI/HIC structure experiments and shadow comparisons
- `baselines/llm_prompting/`: LLM-only baselines
- `docs/`: papers, summaries, and research notes
