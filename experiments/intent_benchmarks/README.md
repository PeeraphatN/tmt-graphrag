# Intent Classification Benchmarks

## Objective
Compare performance of different intent classification strategies:
1. Vector Similarity
2. Local LLM Zero-shot
3. Local LLM Few-shot

## Models
- Qwen 2.5 3B
- Gemma 3 4B
- Phi-4 Mini
- Llama 3.2 1B
- DeepSeek R1 1.5B

## Metrics
- Accuracy
- Latency
- Resource Usage (VRAM/RAM)
